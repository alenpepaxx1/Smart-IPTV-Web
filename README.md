# Smart IPTV WEB

A powerful web-based IPTV player supporting **M3U playlists** and **Xtream Codes API** with a modern, responsive interface.

## ✨ Features
- 📺 Play IPTV channels directly in your browser
- 📂 Support for M3U playlists
- 🔑 Xtream Codes API integration
- 🎨 Modern and responsive UI
- ⚡ Fast loading and smooth playback
- 🌍 Multi-language support (optional)

## 🚀 Getting Started

### Prerequisites
- Node.js (>= 16.x)
- npm or yarn
- A valid IPTV playlist (M3U) or Xtream Codes credentials

### Installation
npm install
npm run dev
